# assessment-task

## How to run this project

1.  Install dependencies:

    ```bash
    npm install
    ```

2.  Run the development server:

    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

3.  Build the project:

    ```bash
    npm run build
    ```

4.  Start the production server:

    ```bash
    npm run start
    ```

5.  Run the linter:

    ```bash
    npm run lint
    ```
